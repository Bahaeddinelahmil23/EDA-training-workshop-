# 📊 EDA Training Workshop

## 📌 Overview

This repository contains training materials for an **Exploratory Data Analysis (EDA) workshop**. The goal of this workshop is to help participants understand how to explore datasets, extract insights, and visualize data effectively using data science tools.

The workshop is divided into multiple sessions, each focusing on a different aspect of EDA learning.

## 📁 Repository Structure

```
EDA-training-workshop/
│
├── session zero/        # Introduction to data science and EDA basics
├── session one/         #  Feature analysis and visualization 
├── session two/         # learning how to use dax
├── session three/       # Data cleaning, preprocessing, and exploration
├── README.md
```

## 🎯 Who Is This For?

* Beginners in data science
* Students learning data analysis
* Participants of the workshop.

## 💡 Learning Objectives

* Understand the EDA pipeline
* Learn how to clean and prepare data
* Practice data visualization
* Extract meaningful insights from datasets
